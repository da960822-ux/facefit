## 메뉴와 페이지 계층

```text
FaceFit
├─ 공개
│  ├─ SC-001 랜딩 /
│  ├─ SC-002 로그인 /login
│  └─ SC-003 회원가입 /signup
├─ 면접 준비
│  ├─ SC-004 새 면접 설정 /onboarding
│  ├─ SC-005 장비 확인 /equipment
│  └─ SC-006 음성 프로필 /voice-profile
├─ 면접
│  ├─ SC-007 면접 시작 안내 /session/live 오버레이
│  └─ SC-008 실시간 면접 /session/live
├─ 결과
│  ├─ SC-009 분석 진행 /analysis
│  ├─ SC-010 결과 리포트 /report
│  └─ SC-011 마이페이지·성장 분석 /dashboard
└─ 비핵심 콘셉트
   └─ SC-012 요금제 /pricing — V1 제외, 결제 없음
```

`/design-preview`는 내부 디자인 검토 라우트로 제품 IA와 화면 ID에서 제외한다.

## 연결 원칙

- SC-004→005→006/007→008→009→010을 주 경로로 한다.
- 음성 프로필은 선택 기능이므로 SC-005에서 SC-007로 직접 이동할 수 있어야 한다.
- SC-009 이탈은 분석 취소가 아니라 SC-011에서 재조회하는 경로다.
- SC-010과 SC-011의 재연습은 기존 설정을 복제할지 새로 입력할지 [협의 필요]다.
- SC-012는 V1 핵심 퍼널과 KPI에서 제외한다.

## 미결정 사항

- SC-004 이후 인증 강제 시점
- 브라우저 뒤로가기 시 설정·세션 데이터 유지 범위
- SC-012의 내비게이션 노출 여부
