## 상세 동작과 예외

| 기능 ID | 정상 시나리오 | 예외 시나리오 | 비즈니스 규칙 | 외부 의존성 | 제외 범위 |
| --- | --- | --- | --- | --- | --- |
| FN-001 | 인증 성공 후 설정 또는 대시보드 이동 | 잘못된 입력·세션 만료 안내 | 계정별 기록 분리 | 인증 제공자 [미정] | 관리자·소셜 인증 확정 |
| FN-002 | PDF/DOCX 검증 후 성공 카드 | 형식·용량·네트워크 실패 후 재시도 | 현행 UI는 최대 10MB 표시, 정책 확정 필요 | 파일 저장·파서 | 이력서 자동 평가 |
| FN-003~005 | 입력과 선택이 요약에 반영 | 필수값 누락 시 이동 차단 | 면접관 1명, 강도 1개 | 질문 생성 정책 | 합격 예측 |
| FN-006~007 | 권한 허용 후 실제 장치 미리보기 | 거부·장치 없음·스트림 중단 | 카메라·마이크 필수 여부 협의 | 브라우저 API | 장치 드라이버 제어 |
| FN-008 | 45초 안내 녹음·확인 후 준비 | 실패 후 재녹음, 건너뛰기 | 선택 기능, 별도 동의 없는 복제 금지 | 녹음·TTS/보이스 제공자 | V1 필수 음성 복제 |
| FN-009 | 첫 진입 안내 후 시작 | 로컬 저장 실패 시 다음 방문 재표시 가능 | 서버 계정 설정으로 간주하지 않음 | localStorage | 별도 온보딩 페이지 |
| FN-010~011 | 면접관 질문→사용자 턴 | 영상 실패 시 음성+정적 화면, 음성 실패 시 텍스트 | 반이중 P0, 인터럽트 P1 | OpenAvatarChat, MuseTalk, TTS, LLM | 완전 자율 채용 면접 |
| FN-012~013 | 수동 완료 후 한 번 저장·전환 | 연타·모달·입력 포커스·전환 중 입력 무시 | 수동 주 방식, VAD 보조 | VAD, ASR, 저장 | VAD만으로 강제 종료 확정 |
| FN-014~015 | 종료 저장→분석→완료 | 저장·분석 실패 후 재시도·재조회 | 분석 중 페이지 이탈 허용 | 분석 오케스트레이션 | 근거 없는 남은 시간 |
| FN-016~019 | 근거 리포트→재연습→성장 비교 | 일부 축 실패·데이터 부족 안내 | 감정·성격·합격 가능성 제외 | CV, ASR, LLM, 리포트 저장 | 비교 집단 임의 생성 |

## 미결정 사항

- FN-001 실제 인증과 FN-002 업로드 계약
- FN-008 음성 프로필의 생성·보존·삭제 정책
- FN-013 자동 완료 확정 조건과 FN-018 설정 복제 범위
