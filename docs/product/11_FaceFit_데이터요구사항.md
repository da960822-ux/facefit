## 데이터 원칙

- DT-002·007·010은 수집 목적과 저장 여부를 별도로 고지한다.
- 실시간 처리에 필요한 데이터와 장기 저장 데이터를 분리한다.
- DT-012는 점수만 저장하지 않고 산식 버전·근거 참조를 함께 관리해야 한다.
- 삭제 시 DT-008을 기준으로 DT-009~014의 연쇄 영향을 검토한다. 물리 DB 방식은 기술 설계에서 결정한다.
- 기술 로그에는 이력서 본문, 원문 답변, 음성·영상, 전체 질문을 기본적으로 기록하지 않는다.

## 미결정 사항

- 전 데이터의 보존 기간과 삭제 완료 기준
- 이력서 원본과 추출 텍스트의 저장 필요성
- 음성 프로필의 모델 파생물·외부 제공자 처리 방식
