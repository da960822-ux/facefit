## 버튼 동작 원칙

- 주요 CTA는 현재 단계의 유효성 검증 후 한 번만 실행한다.
- 비동기 처리 중 버튼은 비활성화하고 처리 상태 문구를 제공한다.
- SC-008의 Space는 입력 요소·버튼·contenteditable 포커스, 모달 열림, 저장 중에는 무시한다.
- 종료·삭제처럼 되돌리기 어려운 동작은 확인과 결과 상태를 제공한다.
- SC-012의 버튼은 안내만 표시하며 결제·구독 요청을 만들지 않는다.

## 미결정 사항

- SC-002~003 인증 성공 후 실제 목적지
- SC-004 필수 입력과 SC-005 면접 시작 차단 기준
- SC-010의 점수 산식·비교 집단 표시 방식
