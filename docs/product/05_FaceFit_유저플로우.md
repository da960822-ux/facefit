```mermaid
flowchart TD

  F -->|"아니요"| E

  G --> H{"권한·장치 정상"}
  H -->|"아니요"| I["복구 안내·재시도"]
  I --> G
  H -->|"예"| J{"음성 프로필"}
  J -->|"등록"| K["SC-006 녹음·확인"]
```

## 답변 턴 상태

```mermaid
stateDiagram-v2
  [*] --> Connecting
  Connecting --> InterviewerSpeaking
  InterviewerSpeaking --> ReadyToAnswer
  ReadyToAnswer --> Recording
  Recording --> Saving: 답변 완료·Space·VAD 후보
  Saving --> InterviewerSpeaking: 다음 질문
  Saving --> EndConfirm: 마지막 질문
  EndConfirm --> Recording: 계속하기
  EndConfirm --> Analyzing: 종료
  Connecting --> Fallback: 아바타 연결 실패
  Fallback --> InterviewerSpeaking: 음성·정적 화면
```

## 미결정 사항

- VAD가 자동 완료를 확정하는지 사용자 확인을 요구하는지 여부
- 재연습 시 이전 설정과 질문을 어느 범위까지 복제할지
- 인증 없이 체험 면접을 허용할지
